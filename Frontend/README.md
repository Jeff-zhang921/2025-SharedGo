# Shared-Go

A real time activity map where users can host and find events/societies near them!

## Getting started

### Prerequisites

Make sure you install these:
- Node.js (https://nodejs.org/) (v18 or later)
- npm
- Git

### Clone repositry

- git clone https://github.com/spe-uob/2025-SharedGo.git
- cd Frontend

### Install dependencies

- npm ci
- npm install react-router-dom

### Run frontend development server

- npm run dev
- Then open the link shown in your terminal

### Build for production

- npm run build

### Preview the production build

- npm run preview
